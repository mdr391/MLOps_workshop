# MLOps_workshop
Google colab code is here https://colab.research.google.com/drive/1kVsMU2OyJj048HBm7cCqbMR7KWxDVxmj?usp=sharing
I also added the file as MLOps_workshop.ipynb.
The dataset diabetic_data.csv has been used from diabetic data on hospital readmission from kaggle.
In the google colab train the model and compared with three different types of models then saved the best model to S3 bucket.

In the AWS, MLFlow was installed. From google colab model was sent to MLFlow with boto3 connectivity and saved the models in S3.
MLFlow screens PDF file have all the screen shots where the artifacts and metrices are visible.

FastAPI code is the main.py.
With dockerfile, requirement.txt, docker-compose.yml and deploy.yml FastAPI was deployed on EC2.
All the screen shots are in Swagger page pdf files.
